# React Table Component with CRUD Operations

Тестовое задание: компонент таблицы на React с возможностью добавления, редактирования и удаления записей.

## 📋 Функциональность

- ✅ Отображение таблицы с колонками: Имя, Дата, Числовое значение, Действия
- ✅ Добавление новых записей через модальное окно с валидацией
- ✅ Редактирование существующих записей
- ✅ Удаление записей с подтверждением
- ✅ Сортировка по всем колонкам с учетом типа данных
- ✅ Поиск по всем ячейкам таблицы
- ✅ Адаптивный дизайн с горизонтальным скроллом на мобильных устройствах
- ✅ Плавные анимации и современный UI

## 🛠 Технологии

- React 18
- Ant Design (UI библиотека)
- CSS3 с анимациями
- dayjs (работа с датами)
- GitHub Pages (деплой)

## 📦 Установка и запуск

```bash
# Клонировать репозиторий
git clone https://github.com/devalmanu/react-table-app.git

# Перейти в папку проекта
cd react-table-app

# Установить зависимости
npm install

# Запустить в режиме разработки
npm start

# Собрать для продакшена
npm run build

# Задеплоить на GitHub Pages
npm run deploy
